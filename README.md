#  Estudos em Linguagem C - Programação I

Este repositório contém o material desenvolvido por mim durante a disciplina de Programação I. O objetivo é documentar minha evolução na lógica de programação e o domínio da sintaxe da linguagem C.

##  Organização do Repositório

*   **[01-Trabalho-Final](./01-Trabalho-Final):** Contém o projeto de conclusão da disciplina. (Sistema de RPG de texto).
*   **[02-Melhores-Algoritmos](./02-Melhores-Algoritmos):** Uma seleção dos algoritmos mais desafiadores, envolvendo manipulação de ponteiros, matrizes e structs.
*   **[03-Exercicios-Diversos](./03-Exercicios-Diversos):** Coleção de mais de 40 exercícios de fixação cobrindo lógica básica, laços de repetição e vetores.
