#  Algoritmos de Destaque (Fundamentos e Lógica)

Esta pasta contém uma seleção de algoritmos desenvolvidos durante a disciplina de Programação I. Eles foram escolhidos por representarem marcos no meu aprendizado, evoluindo da lógica básica para conceitos intermediários de manipulação de memória e estruturas de dados.

##  Destaques da Coleção

Abaixo estão os principais conceitos aplicados nesta seleção:

* **Máquina de Estados Finita (FSM)**: No arquivo `dungeon_crawler_fsm.c`, implementei um sistema de estados para gerenciar o fluxo de um jogo, utilizando `switch-case` para controlar transições complexas.
* **Recursividade**: O algoritmo `inversao_recursiva.c` demonstra o uso de funções que chamam a si mesmas para manipular strings sem o uso de laços iterativos tradicionais.
* **Gerenciamento de Memória Dinâmica**:
    * Uso de `malloc` e ponteiros duplos para criação de matrizes com dimensões definidas pelo usuário.
    * Implementação de vetores expansíveis utilizando `realloc` para ajustar o tamanho da memória em tempo de execução.
* **Estruturas de Dados (Structs)**:
    * **Structs Aninhadas**: Organização de dados complexos (Pessoa > Endereço) para simular um banco de dados estruturado.
    * **Vetores de Objetos**: Manipulação de múltiplos registros de alunos para busca e filtragem de informações.
* **Lógica de Aplicação e Jogos**:
    * Algoritmo para gestão de vagas em estacionamento utilizando mapeamento por matrizes.
    * Uso de sementes de tempo (`time.h`) para geração de números pseudoaleatórios em lógica de jogo.

##  Evolução Técnica
Estes exercícios documentam minha transição da sintaxe básica para a compreensão de como o computador gerencia dados e memória. O domínio desses fundamentos foi essencial para o desenvolvimento do meu **Projeto Final** (localizado na pasta 01).
